Fedora

This is the Fedora logo. Fedora is an open-source, community supported, linux distribution which is sponsored by Red Hat. According to Wikipedia, it was initially released on 6th November 2003 (a few months after I was born!) and has been growing ever since. There are many Fedora Editions, most popular of them being Fedora Workstation and Fedora Server. One of the main differences between them is that Fedora Server doesn't include a desktop environment, but instead depends on management through CLI (command line interface).

Fedora used DNF as a package-manager.

According to linuxtoday.com, FEDORA stands for Flexible Extensible Digital Object Repository Architecture

By default, Fedora Workstation ships using GNOME desktop environment.

Since Fedora is open source, why don't you try to contribute, with anything that you can do! From bug tracking, to designing, or even as a translator! Anyone can find something to do!
